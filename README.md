## if you want to run in localhost

-   Clone this repository

```sh
git clone https://github.com/IlhamM3/profile-react.git
# make sure you are in the folder
cd (move to the folder that was cloned)
```

-   Install package

```sh
pnpm install
```


- Start project
```sh
pnpm run dev
```

- Melakukan build web
```sh
pnpm run build
```
